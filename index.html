Projects Showcase

Smart Helmet HUD

This project integrates an ESP32 microcontroller, MPU6050 sensor, and BLE communication to create a smart bike helmet. It features crash detection, automatic SMS alerting, and turn-by-turn directions received from an Android app using Google Maps API. The minimal OLED display presents essential info to the rider in real time.

8051 Dev Board Validation

This system automates testing for an 8051-based development board. Through UART interfacing with a PC terminal, users can trigger and validate individual peripherals such as buzzers, motors, switches, keypads, and LCDs. It significantly reduces manual testing time in academic and lab environments.
